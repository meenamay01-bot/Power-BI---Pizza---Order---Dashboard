# 🍕 Pizza Order Dashboard – Power BI

## 🍕 Project Overview

This dashboard analyzes pizza sales data across different 
categories, sizes, and time periods. The goal is to help 
restaurant owners identify top-selling pizzas and make 
data-driven decisions to optimize their menu and increase revenue.

## 📊 Key Metrics

| Metric | Value |
|---|---|
| Total Sales | 900K |
| Total Quantity | 50K |
| Total Orders | 48.62K |
| Top Pizza | Classic Deluxe (2453 orders) |

## 📝 Insights from the Dashboard

- **Classic Deluxe Pizza** has the highest order count by pizza type
- **Large (L) size** is most popular with 45.89% of total sales
- **November** shows the highest monthly sales peak
- **Classic category** leads in total revenue
- **XXL size** has the lowest demand with only 1.72% of sales

## 🛠️ Tools Used

- Power BI Desktop
- Microsoft Excel (data source)
- DAX (for calculated measures)
